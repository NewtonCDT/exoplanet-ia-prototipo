Team Newton — Automated detection of exoplanets using NASA Kepler public data.

What this does

This prototype trains a Logistic Regression model on Kepler KOI data to distinguish Confirmed planets from False Positives. It outputs a probability score (pred_proba_confirmed) to prioritize candidates for follow-up.

How to reproduce

  1. Download Kepler KOI Cumulative CSV from NASA Exoplanet Archive.
  2. Place it as data/raw/kepler_koi_full.csv.
  3. Create venv & install deps:
    python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
  4. Train and infer:
    python src/train.py --csv data/raw/kepler_koi_full.csv --out models/model.pkl --metrics reports/metrics.json
python src/infer.py  --model models/model.pkl --csv data/raw/kepler_koi_full.csv --out data/processed/predicciones.csv
